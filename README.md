# WalletService
A microservice for managing user wallets on a payment system

## Assumptions
- User management is outside the scope of this service, hence, user details validations, like phone numbers are not strict.
## To run this locally
- Install Microsoft's Sql Server. See guidelines [here](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
- Update the connection string in the appsettings.json file in the Api project
- Run
